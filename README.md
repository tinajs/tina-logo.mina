# tina-logo.mina
> The Tina.js logo as a mina-component

## Installation
```bash
npm i --save @tinajs/tina-logo.mina
```

## Usage
```html
<config>
{
  "usingComponents": {
    "tina-logo": "~@tinajs/tina-logo.mina"
  }
}
</config>

<template>
  <view>
    <tina-logo />
  </view>
</template>
```
